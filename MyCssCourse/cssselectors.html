CSS Selector Types

  • Universal Selector

    Apply to all type of all HTML element.

    • Element selector

    It selects particular HTML element and apply all of them.

  • Class Selector

    Its a attribute can be added to all of HTML elements to apply them.

  • Attribute Selector

    If element has or has not chosen attribute apply all of HTML element.

  • ID Selector

    Its a attribute can be added to only one Html element to apply it.
